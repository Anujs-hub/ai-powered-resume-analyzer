About This Project


AI Resume Analyzer is a Flask-based web application that analyzes resumes using ATS-based scoring, technical skill detection, resume completeness, keyword matching and Job Description comparison. It helps students improve their resumes before applying for internships and full-time jobs.

Resume Tips


  • Keep your resume limited to one page.
  • Mention your GitHub and LinkedIn profile.
  • Add projects with measurable outcomes.
  • Use keywords according to the Job Description.
  • Include internships, certifications and achievements.
  • Avoid unnecessary graphics to make your resume ATS friendly.
